# TRYITTECH LLP

Premier HR Solutions Provider specializing in Pan-India staffing, offshore IT talent, and workforce solutions.

## Overview
TRYITTECH LLP bridges the gap between global standards and Indian talent, backed by 10+ years of US staffing expertise. We provide scalable workforce solutions and offshore IT talent acquisition.

## Features
- **Staffing Solutions**: Comprehensive pan-India staffing services.
- **IT Talent**: Offshore IT talent acquisition and management.
- **Workforce Management**: Strategic workforce planning and execution.
- **Responsive Design**: Modern, mobile-first UI built with the latest technologies.

## Tech Stack
- **Framework**: [Next.js 15](https://nextjs.org) (App Router)
- **Library**: [React 19](https://react.dev)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
- **Icons**: [Lucide React](https://lucide.dev)

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure
- `src/app`: Next.js App Router pages and layouts.
- `src/components`: UI components (Hero, Navbar, Services, etc.).
- `src/constants`: Application constants and static data.
- `public`: Static assets like images and icons.
